Program Details
---------------

The system is a payroll program for MotorPH that uses CSV files as its data source. It begins with a login page where the user enters a username and password. There are two user roles: employee and payroll staff.

If the user logs in as an employee, the system allows them to enter their employee number and view their basic details such as employee number, name, and birthday.

If the user logs in as payroll staff, the system allows payroll processing for either one employee or all employees. The program reads employee details and attendance records from CSV files, computes the total hours worked for each cutoff, and applies the required rules and conditions when calculating the total working hours. It then calculates the gross salary based on the hourly rate and applies government deductions such as SSS, PhilHealth, Pag-IBIG, and withholding tax. The deductions are based on the combined monthly gross salary and are applied during the second cutoff. The payroll information is displayed for each month from June to December, following the required payroll process flow.
